# Инструкция по настройке аналитики и маркетинга

## 1. Google Analytics 4 (GA4)

### Регистрация:
1. Перейдите на [analytics.google.com](https://analytics.google.com)
2. Нажмите «Начать измерение»
3. Создайте аккаунт и ресурс (Property)
4. Выберите «Веб» → введите URL сайта
5. Получите **Measurement ID** (вида: `G-XXXXXXXXXX`)

### Установка:
Замените в коде сайта `GA_MEASUREMENT_ID` на ваш реальный ID:
```javascript
gtag('config', 'G-XXXXXXXXXX');
```

### Что отслеживается автоматически:
- Просмотры страниц
- Источники трафика
- География посетителей
- Устройства и браузеры
- Время на сайте

## 2. Facebook Pixel

### Регистрация:
1. Перейдите в [Facebook Business Manager](https://business.facebook.com)
2. Events Manager → Подключить источник данных → Веб
3. Назовите пиксель → получите **Pixel ID** (вида: `123456789012345`)

### Установка:
Замените в коде сайта `FB_PIXEL_ID` на ваш реальный ID:
```javascript
fbq('init', '123456789012345');
```

### Что отслеживается:
- PageView (автоматически)
- Конверсии (запись на консультацию)
- Аудитории для ретаргетинга

### Создание аудиторий ретаргетинга:
1. Events Manager → Аудитории
2. Создайте аудиторию «Посетители сайта за 30 дней»
3. Используйте для рекламы в Facebook/Instagram

## 4. Цели и события (Events)

### Рекомендуемые события для отслеживания:

| Событие | Когда срабатывает | Зачем |
|---------|-------------------|-------|
| `contact` | Клик на «Записаться» | Отслеживать заявки |
| `submit_form` | Отправка формы | Конверсия в GA4 |
| `scroll_90` | Прокрутка 90% | Вовлечённость |
| `time_on_page` | 2+ минуты на странице | Качество трафика |

### Как добавить:
В `scripts.js` добавьте:
```javascript
// Отслеживание клика на кнопку «Записаться»
document.querySelectorAll('.btn-primary').forEach(btn => {
    btn.addEventListener('click', () => {
        gtag('event', 'contact', { method: 'button' });
        fbq('track', 'Contact');
    });
});
```

## 5. Проверка работы

### Google Analytics:
- Откройте сайт → GA4 → Realtime (в реальном времени)
- Должен появиться 1 активный пользователь

### Facebook Pixel:
- Установите расширение [Facebook Pixel Helper](https://chrome.google.com/webstore/detail/facebook-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc) для Chrome
- Откройте сайт → значок расширения должен быть зелёным

### Telegram:
- Кликните на плавающую кнопку → должен открыться чат
